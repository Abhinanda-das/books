# About the UAD Console for Apollo Manual


This manual includes operating instructions for UAD Console, Apollo's companion software for Realtime UAD Processing, hardware control, and other functions. Complete Apollo documentation includes the following additional manuals:

<div class="callout callout--info">

**Note:** If you're controlling a Volt 876 audio interface, see the [UAD Console for Volt 876 manual](41288447947284-UAD-Console-for-Volt-876-Manual.html). 

</div>

- Apollo Hardware Manuals – hardware controls, connectors, and specifications for each model.
- Apollo Software Manuals – setup, workflows, and drivers for each connection type (Thunderbolt, USB, and FireWire).
- UAD System Manual – UAD-2 ecosystem and UAD Meter & Control app.
- UAD Plug-Ins Manual – features and functions of all individual UAD-2 plug-ins developed by Universal Audio.

These additional manuals can be found [here](../sections/12262368930324-Apollo-UAD-2-Manuals.html).
