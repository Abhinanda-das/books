# Apollo Desktop USB System Requirements


This article lists the system requirements for the following devices:

- <a href="https://www.uaudio.com/uad-plugins/apollo-solo-usb.html" rel="noopener noreferrer" target="_blank"><strong>Apollo Solo USB</strong></a>
- <a href="https://www.uaudio.com/uad-plugins/apollo-twin-usb.html" rel="noopener noreferrer" target="_blank"><strong>Apollo Twin USB</strong></a>
- <a href="https://www.uaudio.com/uad-plugins/apollo-twin-x-usb.html" rel="noopener noreferrer" target="_blank"><strong>Apollo Twin X USB</strong></a>

## General Requirements

- Windows PC with available built-in USB 3 or 4 port\*

<!-- -->

- Windows 10 64-bit edition or Windows 11

  **Note:** UAD Software v11.8.3 and newer requires Windows 10 April 2018 Update (v1803) and newer. We recommend using the latest available version of Windows 10 for optimal performance.

<!-- -->

- Intel or AMD\* processor
- USB 3.0 or higher cable
- Internet connection to download software and authorize UAD plug-ins
- 25 gigabytes available storage\
  **Note**: We highly recommend having at least 100 GB of free storage on your internal hard drive

## Important Notes

- USB cables must be purchased separately for [Apollo Solo USB](https://www.uaudio.com/uad-plugins/apollo-solo-usb.html) and <a href="https://www.uaudio.com/uad-plugins/apollo-twin-x-usb.html" rel="noopener noreferrer" target="_blank">Apollo Twin X USB</a> models.
- See these articles for additional info on USB cables and connections:
  - <a href="https://help.uaudio.com/hc/en-us/articles/18734594593812-USB-Connection-Info-for-UA-Devices" rel="noopener noreferrer" target="_blank">USB Connection Info for UA Devices</a>
  - <a href="https://help.uaudio.com/hc/en-us/articles/360048570512-Apollo-and-Satellite-Data-Cable-Guide" rel="noopener noreferrer" target="_blank">Apollo USB Cable Guide</a>
- Quad core CPU or better is recommended for optimum DAW performance.
- **Apollo USB models are not compatible with macOS**. For information on macOS-compatible devices, please see the [**Apollo X Series (Thunderbolt)**](360015483551-Apollo-X-Series-System-Requirements.html).

## Windows Usage Notes

- <span style="font-weight: 400;">Certain Windows settings are necessary for optimum performance. See this article for details: </span><a href="https://help.uaudio.com/hc/en-us/articles/27247945568788-Optimizing-Windows-for-UA-Devices" rel="noopener noreferrer" target="_blank"><span style="font-weight: 400;">Optimizing Windows for UA Devices.</span></a>
- <span style="font-weight: 400;">ASIO and WDM driver modes are compatible. Using the ASIO driver is recommended whenever supported by the host application.</span><span style="font-weight: 400;"></span>

## Windows Compatibility Notes

- 2-in-1 systems (notebook/tablet hybrid) are not recommended.
- For information regarding legacy compatibility with older Windows versions, refer to the [UAD Software Archives](215267203-UAD-Software-Archives.html).<span style="font-weight: 400;"></span>
- Windows operating systems running on Virtual Machines are untested and not supported.
- Connect into a USB port that uses an **Intel** USB host controller, or one that complies with **Intel xHCI 1.1** specification or higher. See this article for additional details: <a href="https://help.uaudio.com/hc/en-us/articles/18734594593812-USB-Connection-Info-for-UA-Devices" rel="noopener noreferrer" target="_blank">USB Connection Info for UA Devices</a>. \*
- USB host controllers made by AMD on earlier AM4 platforms may have limited compatibility. See this article for additional details: <a href="https://help.uaudio.com/hc/en-us/articles/18734594593812-USB-Connection-Info-for-UA-Devices#h_01H918V5SEQYW0X8F4EGC8DYX5" rel="noopener noreferrer" target="_blank">PCIe to USB 3 Expansion Cards.</a> \*
- Windows for ARM processors is incompatible with all UA software and hardware.
