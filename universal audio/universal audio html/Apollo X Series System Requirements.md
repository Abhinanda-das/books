# Apollo X Series System Requirements


This article lists the system requirements for the Rack series (**Apollo x6**, **Apollo x8**, **Apollo x8p**, **Apollo x16, and Apollo x16D**) and the Desktop series (**Apollo Twin X, Apollo x4**), both **Gen 1 and Gen 2**.

## General Requirements

- Thunderbolt 3, 4, or 5 cable (not included). See this article for more info: <a href="https://help.uaudio.com/hc/en-us/articles/360048570512-Apollo-and-Satellite-Data-Cable-Guide" rel="noopener noreferrer" target="_blank">Apollo Cable Guide</a>
- Internet connection to download software and authorize UAD plug-ins
- 25 gigabytes of available storage\
  **Note**: We highly recommend having at least 100 GB of free storage on your internal hard drive.

## macOS

- An available Thunderbolt 1, 2, 3, or 4 port

<!-- -->

- <span style="font-weight: 400;">macOS Big Sur 11, Monterey 12, Ventura 13, Sonoma 14, Sequoia 15 and Tahoe 26</span>

<!-- -->

- Thunderbolt 1 and 2 connections require an Apple Thunderbolt 3 to Thunderbolt 2 Adapter (not included)

**Important Note**: Apollo Monitor Correction powered by Sonarworks®** **requires macOS Big Sur 11 as a minimum operating system requirement on Apple computers. 

**Additional Note:** Compatibility with older macOS versions is possible with the 5.9 and 5.2 Legacy versions of SoundID Reference; learn more <a href="https://support.sonarworks.com/hc/articles/5190369966098" rel="noopener noreferrer" target="_blank"><strong>on this page</strong></a>.

## Windows

- Available Thunderbolt 3, 4, or 5 port

<!-- -->

- Windows 10 64-bit edition or Windows 11

  **Note:** UAD Software v11.8.3 and newer requires Windows 10 April 2018 Update (v1803) and newer. We recommend using the latest available version of Windows 10 for optimal performance.

<!-- -->

- Intel or AMD\* CPU

\* *AMD systems are untested with UAD Thunderbolt devices. Although untested systems may work, they are not supported by Universal Audio.*

## Additional Notes

- Operating systems not listed above are untested. Although untested operating systems may work, they are not supported by Universal Audio. Prior UAD software for older operating systems and discontinued UAD products are available in the <a href="http://www.uaudio.com/support/uad/archives.html" rel="noopener noreferrer" target="_blank">UAD software archives</a>.
- Apollo X Gen 2 interfaces require UAD Software v11.5.0 as a minimum version.
- Pre-release and beta versions of operating systems are not supported.
- '2-in-1' systems (notebook/tablet hybrid) are not recommended.
- 'Hackintosh' and Mac computers not made by Apple are not supported.
- For updated details on macOS and Windows compatibility, <a href="https://help.uaudio.com/hc/en-us/articles/210208926-Latest-Mac-Windows-Operating-System-Compatibility-Information" rel="noopener noreferrer" target="_blank">visit this page</a>.
- Windows for ARM processors is incompatible with all UA software and hardware.

## Hardware, License, and OS Requirements for SoundID Reference

- At least one Apollo X interface (Gen 1 or Gen 2) configured as the monitor unit
- UAD Software v11.5.1 or newer
- UAD Console v1.2.1 or newer
- **Apollo Monitor Correction Add-on** product license
- **SoundID Reference** product license(s)
  - SoundID Reference **for Headphones**: Apply headphone profiles
  - SoundID Reference **for Speakers and Headphones**: Apply headphone and speaker profiles (stereo) with calibrated bass management for stereo/subwoofer setups.
  - SoundID Reference **for Multichannel**: Apply headphone and speaker profiles (stereo/multichannel) \
    **Note**: A measurement microphone is needed for speaker calibration.
- An internet connection to authenticate your SoundID Reference license. \
  **Note:** Offline use is not supported.
- Supported operating systems:
  - **macOS 11 Big Sur** and above
  - **Windows 10** and **11**
