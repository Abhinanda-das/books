# DAW Compatibility with UAD DSP Plug-Ins


<span style="font-weight: 400;">UAD DSP plug-ins are compatible with VST, AAX 64, and Audio Units host DAW applications, and are widely used with a variety of major DAW software — including **Pro Tools**, **Logic Pro**, **Cubase**, **Nuendo**, **Live**, and more.</span>

<span style="font-weight: 400;">However, due to plug-in host differences between DAWs — and our own rigorous testing standards — Universal Audio only officially tests the latest UAD software with the operating systems listed below.</span>

#### macOS

- <span style="font-weight: 400;">LUNA (VST3, Audio Units)</span>
- Logic Pro 11.2 <span style="font-weight: 400;">(Audio Units)</span>
- Ableton Live 12 <span style="font-weight: 400;">(VST3, Audio Units)</span>
- Cubase 15 <span style="font-weight: 400;">(VST3)</span>
- <span style="font-weight: 400;">Nuendo 14 (VST3)\*</span>
- Pro Tools 2025 (AAX)
- Studio One 7 <span style="font-weight: 400;">(VST3)\*</span>
- FL Studio 2025 <span style="font-weight: 400;">(VST3)\*</span><span style="font-weight: 400;"></span>

#### Windows

- <span style="font-weight: 400;">LUNA (VST3)</span><span style="font-weight: 400;"></span>
- <span style="font-weight: 400;">Ableton Live 12 (VST3)</span>
- <span style="font-weight: 400;">Cubase 15 (VST3)</span>
- <span style="font-weight: 400;">Nuendo 14 (VST3)\*</span><span style="font-weight: 400;"></span>
- <span style="font-weight: 400;">Pro Tools 2025 (AAX)</span>
- <span style="font-weight: 400;">Studio One 7 (VST3)\*</span>
- <span style="font-weight: 400;">FL Studio 2025 (VST3)</span>

For additional compatibility information, read the <a href="210208926-Latest-macOS-Windows-Operating-System-Compatibility-Information.html" rel="noopener noreferrer" target="_blank"><strong>Latest macOS & Windows Operating System Compatibility Information article</strong></a>.

#### Notes

- UAD software is 64-bit only. 32-bit DAW applications are incompatible.
- The most recent maintenance release of the DAW application is recommended.
- **UAD Software version 11.6.0** require **macOS Big Sur 11 as a minimum operating system on** Apple computers.
- The DAWs marked with an asterisk (**\***) have only been tested in a limited capacity.
