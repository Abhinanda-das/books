# DAW Compatibility with UAD Native Plug-Ins (UADx)


<span style="font-weight: 400;">UAD Native plug-Ins (UADx) are compatible with **VST3**, **Audio Units**, **AAX 64,** and **LUNA** host applications, and are compatible with a variety of major DAWs — including Pro Tools, Logic Pro, Cubase, Nuendo, Live, Studio One, and more.</span>

<span style="font-weight: 400;">However, due to plug-in host differences between DAWs — and our own rigorous testing standards — Universal Audio only officially tests the latest UAD Native plug-ins (UADx) with the DAWs listed below.</span>

#### macOS 

- <span style="font-weight: 400;">LUNA (VST3, Audio Units)</span>
- Logic Pro 11.2 <span style="font-weight: 400;">(Audio Units)</span>
- Ableton Live 12.2 <span style="font-weight: 400;">(VST3, Audio Units)</span>
- Cubase 15 <span style="font-weight: 400;">(VST3)</span>
- Pro Tools 2025 (AAX)
- Studio One Pro 7 <span style="font-weight: 400;">(VST3)</span>
- FL Studio 2025 <span style="font-weight: 400;">(VST3)</span><span style="font-weight: 400;"></span>

#### Windows 

- <span style="font-weight: 400;">LUNA (VST3)</span><span style="font-weight: 400;"></span>
- <span style="font-weight: 400;">Ableton Live 12.2 (VST3)</span>
- <span style="font-weight: 400;">Cubase 15 (VST3)</span><span style="font-weight: 400;"></span>
- <span style="font-weight: 400;">Pro Tools 2025 (AAX)</span>
- <span style="font-weight: 400;">Studio One Pro 7 (VST3)</span>
- <span style="font-weight: 400;">FL Studio 2025 (VST3)</span>

<span style="font-weight: 400;">For additional compatibility information, read the </span><a href="5172379808276-UAD-Native-Plug-ins-UADx-and-UAD-Spark-System-Requirements.html" rel="noopener noreferrer" target="_blank"><strong>UAD Native Plug-ins (UADx) and UAD Spark System Requirements article</strong></a><span style="font-weight: 400;">.</span>

#### <span style="font-weight: 400;">Notes</span>

- <span style="font-weight: 400;">UADx plug-ins are 64-bit only. 32-bit DAW applications are incompatible.</span>
- **UAD Native plug-ins** require **macOS Big Sur 11 as a minimum operating system on** Apple computers.
- <span style="font-weight: 400;">The most recent maintenance release of the DAW application is recommended.</span>
- <span style="font-weight: 400;">Beta versions of DAW applications, macOS, or Windows are not supported.</span>
- <span style="font-weight: 400;">We perform limited testing on **Reaper**, but there are no known issues with this DAW.</span>
