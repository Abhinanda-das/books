# LUNA Keyboard Shortcuts

The following shortcuts allow you to control LUNA with key commands. Note that you can [customize key commands](35314657057172-Customizing-LUNA-Keyboard-Shortcuts.html).

## Windows and Display

- **Toggle Timeline/Mixer:** Command + =

- **Next Window:** Command + \`

- **Previous Window:** Command + Shift + \`

- **Show/Hide Floating Windows:** Shift + W

- **Close Floating Window:** Command + W

- **New Alternate Window:** Shift + Command + = (Opens a second LUNA window)

- **Toggle Full Screen:** Command + Shift + F

- **Spill Selected Bus:** Shift + R

 

## Playback and Timeline

- **Toggle Playback:** Spacebar

- **Toggle Record:** Command + Spacebar (or Numeric Keypad 3)

- **Restart Playback:** Control + Option + Command + Spacebar

- **Loop Playback:** Control + L (or Numeric Keypad 4)

- **Return To Zero:** Return

- **Toggle Pre/Post Roll:** Command + K

- **Toggle Metronome:** K (or Numeric Keypad 7)

- **Toggle Count In:** Shift + K (or Numeric Keypad 8)

- **Create Marker:** Enter

 

## Zooming and Scrolling

- **Auto Scroll:** Shift + A

- **Zoom In:** Command + \] (or T)

- **Zoom Out:** Command + \[ (or R)

- **Frame Selection:** E

- **Increase Selected Track Heights:** Control + ↑

- **Decrease Selected Track Heights:** Control + ↓

- **Increase All Track Heights:** Control + Option + ↑

- **Decrease All Track Heights:** Control + Option + ↓

- **Zoom In Audio Waveforms:** Option + Command + \]

- **Zoom Out Audio Waveforms:** Option + Command + \[

- **Reset Audio Waveform Zoom:** Control + Option + Command + \[

- **Scroll To Left Selection:** Shift + ← (or Q)

- **Scroll To Right Selection:** Shift + → (or W)

- **Scroll Mixer (One Track Left/Right):** ← / →

- **Scroll Mixer (One Page Left/Right):** Command + ← / →

- **Scroll Mixer (To First/Last Track):** Option + Command + ← / →

- **Scroll Mixer (Up/Down):** ↑ / ↓

 

## Record / Input / Solo / Mute

- **Toggle Record Arm:** Shift + R

- **Toggle Input Enable:** Shift + T

- **Toggle Input Enable on All Record-Enabled Tracks:** Option + K

- **Toggle Solo:** Shift + S

- **Toggle Mute:** Shift + M

- **Toggle Record:** Command + Spacebar (or Numeric Keypad 3)

- **Discard Recording:** Shift + Command + . (Period)

 

## Editing and Fades

- **Clear:** Delete

- **Undo:** Command + Z

- **Redo:** Command + Shift + Z

- **Cut:** Command + X (or X)

- **Copy:** Command + C (or C)

- **Paste:** Command + V (or V)

- **Duplicate Selection:** Command + D

- **Shift Cut:** Shift + X

- **Shift Paste:** Shift + V

- **Shift Duplicate:** Shift + D

- **Shift Insert Time:** Shift + I

- **Shift Clear:** Shift + Delete

- **Mute Selection:** Command + M

- **Create Fades:** Command + F

- **Fade To Clip Start (Fade In):** D

- **Fade To Clip Stop (Fade Out):** G

- **Trim From Clip Start:** A

- **Trim From Clip End:** S

- **Trim Clip to Selection:** Command + T

- **Separate Selection:** Command + E (or B)

- **Consolidate Selection:** Option + Shift + 3

- **Reverse Selection:** Control + Shift + R

- **Heal Separation:** Command + H

- **Export Mixdown:** Option + Command + B

- **Export Clips:** Command + Shift + K

- **Generate Sinewave in Selection:** Control + Option + Shift + 3 (or Control + Option + Shift + \#)

 

## Making Selections

- **Select All:** Command + A

- **Select All Tracks:** Shift + Command + A

- **Set Selection Start (while stopped or playing back):** ↓

- **Set Selection End (while playing back):** ↑

 

## Moving Selections

- **Move Selection Up:** P

- **Move Selection Down:** ;

- **Move Selection to Next Bar:** \] (or Numeric Keypad 2)

- **Move Selection to Previous Bar:** \[ (or Numeric Keypad 1)

- **Move Selection To Next Clip Edge:** '

- **Move Selection To Previous Clip Edge:** L

- **Move Selection To Previous Transient:** Option + Tab (or Option + L)

- **Move Selection To Next Marker:** Control + Option + '

- **Move Selection To Previous Marker:** Control + Option + L

- **Loop Select:** Command + Shift + L

- **Extend Selection to Previous Marker:** Shift + Control + Option + L

- **Extend Selection To Session Start:** Shift + Return

- **Extend Selection To Session End:** Option + Shift + Return

# LUNA Menu Reference

The following menu items are available in LUNA. Note that not all menu items have key shortcuts. 

 

## LUNA menu

- **About LUNA**: Shows the LUNA version information.

- **Check for Updates...**: Checks the server for LUNA updates.

- **Settings (Command+,)**: Opens the LUNA settings page.

- **Hide LUNA (Command+H)**: Hides LUNA windows.

- **Hide Others (Option+Command+H)**: Hides everything except LUNA windows.

- **Quit LUNA (Command+Q)**: Quits LUNA.

 

## File menu

- **New (Command+N)**: Open the LUNA New Session window.

- **Open... (Command+O)**: Open a Finder window to open a session.

- **Open Recent**: Allows you to choose from a list of recent sessions to open.

- **Open Recent \> Browse All Recent (Option+Command+O)**: Opens the Recent Session browser.

- **Open Recent \> Clear Recent List**: Clears the Recent Sessions List.

- **Open Version (Shift+Command+O)**: Open a session version, bookmark, or auto-backup.

- **Close (Command+W)**: Closes the topmost floating window in the session.

- **Close Session (Shift+Command+W)**: Closes the session.

- **Save Bookmark (Command+S)**: Save the state of the session under the current version.

- **Save New Version (Shift+Command+S)**: Save a new version within the session.

- **Duplicate**: Saves a copy of the session with a new name.

- **Rename**: Opens the Rename popover so you can rename the session.

- **Revert To**: Shows a list of automatically saved session bookmarks.

- **Import... (Command+I)**: Opens a Finder window to select audio or MIDI files.

- **Import Session... (Option+I)**: Opens the Import Session Data window.

- **Export \> Mixdown (Option+Command+B)**: Opens the Export Mixdown dialog.

- **Export \> All Tracks as Files**: Exports all tracks as separate files.

- **Export \> Multitrack MIDI File**: Saves a multitrack MIDI file from the session.

- **Export \> Clips (Shift+Command+K)**: Opens the Export Clips dialog.

 

## Edit menu

- **Undo (Command+Z)**: Undo anything except a change made during recording.

- **Redo (Shift+Command+Z)**: Redo anything except a change made during recording.

- **Cut (Command+X)**: Cuts a selection to the clipboard.

- **Copy (Command+C)**: Copies a selection to the clipboard.

- **Paste (Command+V)**: Pastes a selection.

- **Clear (Delete)**: Deletes a selection.

- **Duplicate (Command+D)**: Pastes a selection right after the current selection.

- **Consolidate (Option+Shift+3)**: Combines two or more clips into one.

- **Reverse (Control+Shift+R)**: Reverses a selection or clip.

- **Select All (Command+A)**: Selects all clips on a timeline track.

- **Shift \> Cut (Shift+C)**: Cuts a selection and shifts clips to fill the space.

- **Shift \> Clear (Shift+Delete)**: Deletes selection and shifts clips to fill space.

- **Shift \> Paste (Shift+V)**: Pastes and shifts clips to accommodate data.

- **Shift \> Duplicate (Shift+D)**: Duplicates and shifts clips to accommodate data.

- **Shift \> Insert Time (Shift+I)**: Inserts space of the current selection length.

- **Nudge \> Left (,)**: Nudges selection to the left by the nudge amount.

- **Nudge \> Left Next Amount (M)**: Nudges left by the next higher nudge amount.

- **Nudge \> Right (.)**: Nudges selection to the right by the nudge amount.

- **Nudge \> Right Next Amount (/)**: Nudges right by the next higher nudge amount.

- **Nudge \> To Next Clip (Control+Option+.)**: Nudges clip to the boundary of the next clip.

- **Nudge \> To Previous Clip (Control+Option+,)**: Nudges clip to the boundary of the previous clip.

- **Nudge \> Increment Nudge Value (Option+Command+=)**: Increments the nudge amount.

- **Nudge \> Decrement Nudge Value (Option+Command+-)**: Decrements the nudge amount.

- **Nudge Content \> Left (Control+,)**: Nudges content within the clip to the left.

- **Nudge Content \> Right (Control+.)**: Nudges content within the clip to the right.

- **Mute Selection (Command+M)**: Mutes a clip.

- **Separate Selection (Command+E)**: Separates a clip at the playhead location.

- **Separate on Grid**: Separates selection/clip into clips on every grid line.

- **Heal Separation (Command+H)**: Recombines selected separated clips.

- **Create Fades (Command+F)**: Creates a fade or crossfade.

- **Delete Fades**: Deletes fades in selected audio.

- **Fade to Clip Start (D)**: Fade in from playhead to start.

- **Fade to Clip Stop (G)**: Fade out from playhead to end.

- **Trim From Clip Start (A)**: Trims from clip start to the playhead.

- **Trim From Clip End (S)**: Trims from clip end to the playhead.

- **Trim Clip to Selection (Command+T)**: Trims the clip to the selection.

- **Increase Grid (Shift+=)**: Increases grid setting by one increment.

- **Decrease Grid (Shift+-)**: Decreases grid setting by one increment.

- **Quantize (Command+U)**: Quantizes notes based on current settings.

- **Quantize Settings (Shift+Command+U)**: Opens the Quantize settings browser.

- **Set Bar One**: Sets the current playhead position as bar one.

- **Align Bar (Shift+Option+Command+I)**: Aligns playhead to bar and adds tempo event.

- **Align Grid (Option+Command+I)**: Aligns playhead to grid and adds tempo event.

 

## Track menu

- **New Tracks (Shift+Command+N)**: Open the Create New Tracks dialog.

- **New Instrument (Shift+Command+I)**: Creates a new instrument track with Shape.

- **New Track from Preset (Shift+Command+P)**: Creates tracks from a Preset.

- **Save Track as Preset (Option+Command+P)**: Saves selected tracks as a Preset.

- **Selection Grouping (Control+G)**: Enables/disables Selection Grouping.

- **Suspend Groups (Shift+Command+G)**: Turns all track groups on or off.

- **Duplicate (Option+D)**: Duplicates selected tracks with content.

- **Duplicate Without Content (Shift+Option+D)**: Duplicates tracks without content.

- **Bounce (Command+B)**: Bounces an entire track or tracks.

- **Bounce Range (Option+B)**: Bounces the selected range of a track.

- **Bounce Settings (Option+Shift+B)**: Shows bounce settings.

- **New Version \> On All Tracks (Control+Option+)**: Creates a new Version on all tracks.

- **New Version \> On Selected Tracks (Control+)**: Creates a new Version on selected tracks.

- **Delete Selected Tracks (Shift+Command+Delete)**: Deletes all selected tracks.

- **Hide Selected Tracks**: Hides all selected tracks.

- **Find Track (Option+Command+F)**: Opens Tracks Focus Browser Search.

 

## Mixing menu

- **Solo Mode \> Latching**: Allows you to solo multiple tracks.

- **Solo Mode \> Exclusive**: Only one element can be soloed at a time.

- **Create Bus (Shift+Command+B)**: Creates a bus.

- **Assign Console Extension**: Opens Session Console Assignment browser.

- **Convert UAD-2 Plug-Ins to UADx**: Converts DSP plug-ins to native.

- **Convert UADx Plug-Ins to UAD-2**: Converts native plug-ins to DSP.

 

## Assistant menu

- **Assistant**: Enables the LUNA Assistant features.

- **Instrument Detection**: Toggles Instrument Detection on/off.

- **Instrument Detection Settings…**: Set options for colors and naming.

- **Run Instrument Detection**: Runs detection on current session.

- **Voice Control**: Toggles Voice Control feature on/off.

- **Voice Control Settings**: Specify wake phrase and input.

- **Extract Tempo from Selection**: Extracts tempo from Timeline selection.

- **Extract Tempo from Entire Session**: Extracts tempo from entire session.

 

## Transport menu

- **Loop Playback (Control+L)**: Enables looping for playback/record.

- **Play From Stop Location**: Playhead stays at stop location instead of returning.

- **Pre-Roll**: Plays session before the playhead/selection.

- **Post-Roll**: Plays session after the selection ends.

- **Toggle Pre/Post-Roll (Command+K)**: Toggles pre/post-roll on/off.

- **MIDI Merge (Numeric Keypad 9)**: Toggles MIDI Merge and Replace modes.

- **All Notes Off**: Stops all MIDI messages.

- **Accelerated Realtime Monitoring**: Toggles ARM mode for low latency.

- **Console Tracking Mode**: Console inputs remain live/unmuted.

- **Toggle Playback (Spacebar)**: Starts or stops playback.

- **Toggle Record (Command+Spacebar)**: Toggles recording.

- **Restart Playback (Control+Option+Command+Spacebar)**: Restarts from original position.

- **Discard Recording (Command+Shift+.)**: Stops and discards current recording.

- **Loop Ruler \> Update Loop (Command+L)**: Matches Loop ruler to Timeline selection.

- **Loop Ruler \> Loop Select (Shift+Command+L)**: Matches Timeline to Loop selection.

 

## Navigation menu

- **Zoom In (Command+\])**: Zoom in horizontally.

- **Zoom Out (Command+\[)**: Zoom out horizontally.

- **Frame Selection (E)**: Fills window with selection; press again to revert.

- **Return to Zero (Return)**: Moves playhead to session start.

- **Go to End**: Moves playhead to end of last clip.

- **Zoom In Audio (Option+Command+\])**: Zooms waveforms in horizontally.

- **Zoom Out Audio (Option+Command+\[)**: Zooms waveforms out horizontally.

- **Reset Audio Zoom (Control+Option+Command+\[)**: Resets horizontal zoom.

- **Selection \> Move Up (P)**: Moves selection to track above.

- **Selection \> Move Down (;)**: Moves selection to track below.

- **Selection \> Move to Next Transient (Tab)**: Moves playhead to next transient.

- **Selection \> Move to Previous Transient (Option+Tab)**: Moves to previous transient.

- **Selection \> Move to Next Bar (\])**: Moves playhead to next bar.

- **Selection \> Move to Previous Bar (\[)**: Moves playhead to previous bar.

- **Selection \> Move to Next Clip Edge (‘)**: Moves playhead to next clip edge.

- **Selection \> Move to Previous Clip Edge (L)**: Moves playhead to previous clip edge.

- **Track Heights \> Increase Selected (Control+↑)**: Increases height of selected tracks.

- **Track Heights \> Decrease Selected (Control+↓)**: Decreases height of selected tracks.

- **Auto-Scroll (Shift+A)**: Scrolls timeline automatically during playback.

 

## View menu

- **Section \> Browser**: Shows/hides the Focus Browser.

- **Section \> Focus Channel**: Shows Focus channel in Timeline.

- **Section \> Track Versions**: Shows Versions panel in track controls.

- **Section \> Monitor**: Shows Monitor section at the right.

- **MIDI Keyboard Mode (Option+Command+M)**: Toggles typing keyboard MIDI input.

- **Toggle Timeline/Mixer (Command+=)**: Switches between Timeline/Mixer views.

 

## Windows menu

- **New Alternate Window (Shift+Command+=)**: Opens a second LUNA window.

- **Previous Window (Shift+Command+’)**: Cycles to the previous window.

- **Next Window (Command+’)**: Cycles to the next window.

- **Show/Hide Floating Windows (Shift+W)**: Shows/hides floating windows.

- **Full Screen (Shift+Command+F)**: Toggles full screen mode.

 

## Help menu

- **Search**: Search for items in the menus.

- **LUNA Documentation**: Links to online documentation.

- **Give Feedback...**: Opens the Feedback form.

- **Show Keyboard Shortcuts**: Opens a list of keyboard shortcuts.

- **Tooltips**: Toggle between Off, On, and Help Mode.
