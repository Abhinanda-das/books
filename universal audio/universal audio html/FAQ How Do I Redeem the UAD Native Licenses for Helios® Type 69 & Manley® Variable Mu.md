# FAQ: How Do I Redeem the UAD Native Licenses for Helios® Type 69 & Manley® Variable Mu?


Since February 27, 2024, <a href="https://www.uaudio.com/uad-plugins/channel-strips/helios-type-69-preamp-eq.html" rel="noopener noreferrer" target="_blank"><strong>Helios® Type 69 Preamp & EQ Collection</strong></a> and the <a href="https://www.uaudio.com/uad-plugins/compressors-limiters/manley-variable-mu.html" rel="noopener noreferrer" target="_blank"><strong>Manley® Variable Mu Limiter Compressor</strong></a> plug-ins are now available in the **UAD Native** format. Those who already own the **UAD DSP licenses** are entitled to claim the **UAD Native licenses** for only **\$2.99** each by following the steps below:

1.  If you are entitled to this offer, navigate to the respective product pages and select the **Upgrade** option to add to your cart. In the screenshot below, we've navigated to the <a href="https://www.uaudio.com/uad-plugins/channel-strips/helios-type-69-preamp-eq.html" rel="noopener noreferrer" target="_blank"><strong>Helios® Type 69 Preamp & EQ Collection</strong></a> product page and selected the **Upgrade** option after confirming that the UAD DSP license was already owned. If you already own the <a href="https://www.uaudio.com/uad-plugins/compressors-limiters/manley-variable-mu.html" rel="noopener noreferrer" target="_blank"><strong>Manley® Variable Mu Limiter Compressor</strong></a> UAD DSP license, the process will be similar.\
    ![2_Helios_Upgrade.png](FAQ%20How%20Do%20I%20Redeem%20the%20UAD%20Native%20Licenses%20for%20Helios%C2%AE%20Type%2069%20%26%20Manley%C2%AE%20Variable%20Mu_assets/c48346b8eb180d8e1340d1c7811cddad2db3bb9b.png)
2.  Click the **Add to Cart** button to add the **Upgrade** license to your cart, then go through the checkout process when you are ready to make the purchase.
3.  On the checkout page, the **\$2.99** price should be reflected to indicate you are purchasing the **UAD Native** license upgrade.\
    ![3_Helios_CheckoutCart.png](FAQ%20How%20Do%20I%20Redeem%20the%20UAD%20Native%20Licenses%20for%20Helios%C2%AE%20Type%2069%20%26%20Manley%C2%AE%20Variable%20Mu_assets/52f8e83d57e3dfe4c7cd862a732bcb1a8ea1b117.png)
4.  Click the **Checkout** button at the bottom of the page to go through with the purchase and enter the requested payment information.
5.  When the purchase is completed, download your plug-in(s) via **UA Connect**. For more information about UA Connect and downloading **UAD Native plug-ins**, read <a href="https://help.uaudio.com/hc/en-us/articles/5172589083028" rel="noopener noreferrer" target="_blank"><strong>this article</strong></a>.
