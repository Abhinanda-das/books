# LUNA System Requirements


## General Requirements

- <span style="font-weight: 400;">macOS Big Sur 11, Monterey 12, Ventura 13, Sonoma 14, Sequoia 15 and Tahoe 26</span>

<!-- -->

- Windows 10 64-bit edition or Windows 11

  **Note:** UAD Software v11.8.3 and newer requires Windows 10 April 2018 Update (v1803) and newer. We recommend using the latest available version of Windows 10 for optimal performance.

For more information about LUNA on Windows, <a href="https://help.uaudio.com/hc/en-us/articles/26636829359252-FAQ-LUNA-for-Windows" rel="noopener noreferrer" target="_blank">read this article</a>.

- 600 MB available storage on system volume

- Other required software:

  - [UA Connect](https://www.uaudio.com/pages/download-uad-apollo) – to install and authorize LUNA (UA Connect must remain installed) 

  - iLok License Manager – included with UA Connect

  - Free [PACE iLok](https://www.ilok.com/) account – created for you when installing UA Connect

## System Recommendations

- Intel quad-core i7 (or better) or Apple silicon processor recommended

- 16 GB RAM or more is recommended

- SSD storage for LUNA sessions is recommended

- Rotational storage for LUNA sessions may reduce performance

- Rotational storage below 7200 RPM is unsupported

- Rotational external storage must be USB 3.0 or faster

## SSD Storage Requirements

- SSD drive format:

  - macOS: APFS ([Apple File System](https://support.apple.com/guide/disk-utility/add-delete-or-erase-apfs-volumes-dskua9e6a110/22.0/macOS))
  - Windows: NTFS ([New Technology File System](https://learn.microsoft.com/en-us/windows-server/storage/file-server/ntfs-overview))
  - ExFAT, FAT, macOS OS Extended, and macOS Fusion hybrid formats are unsupported

- External SSD must be within USB 3.0, USB 3.1, PCIe, or Thunderbolt enclosure

## UAD Instruments Requirements

- Sample-based UAD instruments must be installed on SSD volume
- 20 MB to 10 GB available SSD storage (varies per UAD Instrument)
- Individual storage requirements are listed on UA Connect's LUNA page

## Apollo Mode Requirements

- UAD Software v11.7 or newer – get via <a href="https://www.uaudio.com/pages/download-uad-apollo" rel="noopener noreferrer" target="_blank">UA Connect</a>

- Thunderbolt-equipped Apollo interface connected via built-in Thunderbolt port

- All system requirements for Thunderbolt-equipped Apollo models still apply with LUNA

- First-generation (silver) Apollo rackmount models equipped with any Universal Audio [Thunderbolt Option Card](https://help.uaudio.com/hc/en-us/articles/360061415251) and connected via built-in Thunderbolt can be used in Apollo Mode

- All UA Thunderbolt Option Card models (Thunderbolt 1, 2, and 3 cards) are supported
