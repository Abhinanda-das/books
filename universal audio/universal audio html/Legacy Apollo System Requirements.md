# Legacy Apollo System Requirements


# Thunderbolt 2 System Requirements

The requirements for the Thunderbolt 2 devices apply to the following models:

- **Apollo Twin**
- **Apollo Twin MkII**
- **Apollo FireWire**
- **Apollo 8**
- **Apollo 8p**
- **Apollo 16**
- **Apollo 16 MkII**

## General Requirements

- Thunderbolt 2 cable (not included).
- Internet connection to download software and authorize UAD plug-ins.
- 100 gigabytes available storage minimum.

## macOS

- Available Thunderbolt 1, 2, 3, 4, or 5 port.

<!-- -->

- <span style="font-weight: 400;">macOS Big Sur 11, Monterey 12, Ventura 13, Sonoma 14, Sequoia 15 and Tahoe 26</span>

**Note**: **UAD Software version 11.6.0** requires **macOS Big Sur 11 **as a minimum operating system on Apple computers.

## Windows

- Available Thunderbolt 3 port, with the [Thunderbolt 2 option card](https://help.uaudio.com/hc/en-us/articles/360061415251-Apollo-Thunderbolt-Option-Card-Info) installed in the Apollo.

<!-- -->

- Windows 10 64-bit edition or Windows 11

  **Note:** UAD Software v11.8.3 and newer requires Windows 10 April 2018 Update (v1803) and newer. We recommend using the latest available version of Windows 10 for optimal performance.

<!-- -->

- Qualified Thunderbolt 3 to Thunderbolt adapter (not included).\
   

# FireWire System Requirements

## General Requirements

- Windows 10 64-bit edition or Windows 11 64-bit edition on PC.
- macOS Catalina 10.15 on Apple computers.
- Available FireWire 800 port.
- FireWire 800 cable (included).
- Internet connection to download software and authorize UAD plug-ins.
- 100 gigabytes available storage.
- On Windows systems, a qualified **PCIe-to-FireWire adapter card** is required.

 

## Notes 

- **macOS:** Thunderbolt 3 or 4 connections require an Apple Thunderbolt 3 to Thunderbolt 2 Adapter when used with Apollo Thunderbolt 1 or 2 option cards.
- **Windows:** Thunderbolt 4 is not backwards compatible with Thunderbolt 2.
- **Thunderbolt 3 option cards** are not tested or supported with silver Apollo interfaces.
- **Thunderbolt 1 Apollo option cards** are not compatible with Windows.
- Operating systems not listed above are untested. Although untested operating systems may work, they are not supported by Universal Audio. Prior UAD software for older operating systems and discontinued UAD products is available in the <a href="http://www.uaudio.com/support/uad/archives.html" rel="noopener noreferrer" target="_blank">UAD software archives</a>. 
- Pre-release and beta versions of operating systems are not supported.
- <span style="box-sizing: border-box; transition-duration: 0.01ms !important; animation-duration: 0.01ms !important; animation-iteration-count: 1 !important; scroll-behavior: auto !important; margin-bottom: 0px;">AMD systems are untested with UAD Thunderbolt devices. Although untested systems may work, they are not supported by Universal Audio.</span>
- <span style="font-weight: 400;">2-in-1 systems (notebook/tablet) are not recommended.\
  </span>
- <span style="font-weight: 400;">'Hackintosh' and 'Mac computers' not made by Apple are not supported.</span>
- For updated details on macOS and Windows compatibility, <a href="https://help.uaudio.com/hc/en-us/articles/210208926-Latest-Mac-Windows-Operating-System-Compatibility-Information" rel="noopener noreferrer" target="_blank">read this article</a>.
