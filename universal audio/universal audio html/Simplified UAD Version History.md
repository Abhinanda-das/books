# Simplified UAD Version History


This simplified version history list is intended to be used to determine which UAD software versions and plug-ins are compatible with your operating system, however additional details that may be important to your particular system (such as bug fixes, known issues, DAW compatibility, etc) are not listed here. For complete details for each UAD software release, please see the <a href="http://www.uaudio.com/support/uad/versions.html" rel="noopener noreferrer" target="_blank">UAD Version History & Release Notes</a>. 

Click on any of the UAD plug-in names below to see the **minimum UAD software version and operating systems required for compatibility**.

Any UAD plug-ins not listed here were released in UAD v7.2 or below. See the <a href="http://www.uaudio.com/support/uad/versions.html" rel="noopener noreferrer" target="_blank"><strong>UAD Version History & Release Notes</strong></a> for more information on those plug-ins and software versions.

If you are using an older operating system and need to download an older version of UAD software for compatibility, see the <a href="http://www.uaudio.com/support/uad/archives.html" rel="noopener noreferrer" target="_blank">UAD Software Archives</a>.
