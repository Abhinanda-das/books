# Sphere System Requirements


## Hardware

- Sphere LX, DLX, or L22 microphone
- Two mic preamps with balanced XLR inputs and +48V phantom power

## macOS

- Apple silicon or Intel processor

<!-- -->

- <span style="font-weight: 400;">macOS Big Sur 11, Monterey 12, Ventura 13, Sonoma 14, Sequoia 15 and Tahoe 26</span>

## Windows

- Intel or AMD processor

<!-- -->

- Windows 10 64-bit edition or Windows 11

  **Note:** UAD Software v11.8.3 and newer requires Windows 10 April 2018 Update (v1803) and newer. We recommend using the latest available version of Windows 10 for optimal performance.

## Available plug-in formats

- Native (CPU processing):\
  • VST2\
  • VST3\
  • AAX\
  • Audio Units (AU)

<!-- -->

- DSP (realtime processing):\
  • UAD: VST3, VST2, AAX, Audio Units\
  • HDX: AAX
