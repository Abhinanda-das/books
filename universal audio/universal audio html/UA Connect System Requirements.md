# UA Connect System Requirements


This article lists all of the current system requirements for <a href="https://www.uaudio.com/pages/download-ua-connect" rel="noopener noreferrer" target="_blank"><strong>UA Connect</strong></a>**.**

- <span style="font-weight: 400;">macOS Big Sur 11, Monterey 12, Ventura 13, Sonoma 14, Sequoia 15 and Tahoe 26</span>

<!-- -->

- Windows 10 or Windows 11 (64-bit editions)
- Intel, AMD, or Apple silicon processor
- An internet connection to download software

#### Notes

- Operating systems not listed above are untested. Although untested operating systems may work, they are not supported by Universal Audio.
- Pre-release and beta versions of operating systems are not supported.
- Windows running via Bootcamp on Mac systems is not supported.
- 'Hackintosh' and 'Mac computers' not made by Apple are not supported.
