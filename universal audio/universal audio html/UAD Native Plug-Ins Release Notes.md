# UAD Native Plug-Ins Release Notes


This article lists all release notes and changes related to UAD Native plug-in updates.

## May 2026

### New

- UAD Enigmatic '82 Overdrive Special Amp Plug-in

## April 2026

#### New

- Little Labs VOG Bass Resonance Plug-In

## March 2026

#### Improved

##### Paradise Guitar Studio

- Various minor fixes and improvements

##### Sound City Studios

- Default preset has been refined
- Shift+click modifier retains current Distance when changing mics
- Improved microphone interactions in Studio View
- Added optional guided tips to more controls
- Various additional minor fixes and improvements

## January 2026

#### New

- UA 610 Tube Preamp & EQ Plug-In Collection

#### Improved

##### Paradise Guitar Studio

- Pedals can be removed from the pedal board with a single click
- Overload indicator added to input meter
- Default output level matches factory presets
- Section gain sliders are easier to adjust
- Long control names are visible in automation assignment overlay
- NKS support 
- Various additional minor fixes and improvements 

##### Ocean Way Studios Deluxe 

- Mixer effects are no longer affected by Mix control in Reverb Mode
- 110 EQ polarity is no longer inverted
- Pre-equalization for each Chamber to reduce rumble
- Added hover tips to more controls
- Various additional minor fixes and improvements

## December 2025

#### New

- Paradise Guitar Studio Plug-In

#### Improved

##### A-Type Multiband 

- Switch automatically between native and DSP versions in LUNA

## November 2025

#### New

- Ocean Way Studios Deluxe Plug-In

## October 2025

#### New

- NKS Support for all UAD Native Plug-Ins (UADx)

#### Improved

- Improved plug-in scanning with Ableton Live and Pro Tools 2025.6
- **Waterfall B3 Organ** - Resolved issue preventing Multi-Channel MIDI control in VST3

## September 2025

#### New

- UAD Showtime '64 Tube Amp
- Resolved Audio Units error during Logic Pro offline bounce that could occur with UAD guitar amp plug-ins 

## June 2025

#### New

- A-Type Multiband Dynamic Enhancer
- Anthem Analog Synthesizer Plug-In
- MIDI Learn
- Performance optimizations

## April 2025

#### New

- UAD Woodrow ‘55 Instrument Amp Plug-in

#### Compatibility Changes

- Support for **newest DAWs**
  - **LUNA 1.8**
  - **Logic Pro 11.1**
  - **Ableton Live 12.1**
  - **Cubase 14**
  - **Pro Tools 2024.10**
- macOS Big Sur 11, Monterey 12, Ventura 13, Sonoma 14, and Sequoia 15 are compatible
- Windows 10 and 11 are compatible

## March 2025

#### New

- Native UADx plug-ins can now be activated on up to three devices, including <a href="35180558198548-FAQ-How-to-Activate-UAD-Plug-ins-and-Apps-on-Your-Computer.html" rel="noopener noreferrer" target="_blank">local host (machine) locations</a>

## February 2025

#### New

- UA 1176 Classic FET Compressor
- Support for Apple M4 processors

## November 2024

#### New 

- Support for **macOS Sequoia 15**
- Support for **newest DAWs**
  - **Logic Pro 11.1**
  - **Ableton Live 12.1**
  - **Cubase 14**
  - **Pro Tools 2024.10**
  - **Studio One Pro 7**
  - **FL Studio 2024**

#### Improved

- **Topline Vocal Suite** – Tuner pitch tracking is more consistent on sustained notes
- **Topline Vocal Suite** – Shift-click in navigation bar to toggle modules on/off
- **Topline Vocal Suite** – New Logic Pro sessions use correct key/scale defaults
- **Dream, Lion, Ruby** – Input gain dB value is displayed when mouse is hovered over knob
- **Ruby** – Resolved noises that could occur when changing presets, channel, and boost type
- **Dream** – Output polarity now matches input polarity
- **Lion, Ruby** – Latency is reduced at all sample rates
