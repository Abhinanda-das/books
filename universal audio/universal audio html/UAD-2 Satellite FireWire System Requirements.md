# UAD-2 Satellite FireWire System Requirements


## General Requirements

<span style="color: #C72809;">**Important Note**</span>**: UAD Software 11.6 and later are incompatible with UAD devices connected by FireWire. **\
UAD Software 11.5.2 and earlier with FireWire compatibility are available in the [UAD Software Archives](215267203-UAD-Software-Archives.html)

- Windows 10 64-bit edition or Windows 11 on PC.
- macOS Catalina 10.15 on Apple computers.\
  **Important note**: macOS Catalina 10.15 is the last OS on Apple machines compatible with UAD devices connected by FireWire. Subsequent macOS versions have officially dropped the FireWire protocol altogether.
- Available FireWire 800 port.
- FireWire 800 cable (included).
- Internet connection to download software and authorize UAD plug-ins.
- Compatible plug-in host DAW software.
- 25 gigabytes available storage.\
  **Note**: We highly recommend having at least 100 GB of free storage on your internal hard drive.

## Notes

- Operating systems not listed above are untested. Although untested operating systems may work, they are not supported by Universal Audio. Prior UAD software for older operating systems and discontinued UAD products is available in the <a href="http://www.uaudio.com/support/uad/archives.html" rel="noopener noreferrer" target="_blank">UAD software archives</a>.
- On **Windows** systems, a qualified **PCIe-to-FireWire adapter card** is required.
- Pre-release and beta versions of operating systems are not supported.
- 'Hackintosh' and 'Mac computers' not made by Apple are not supported.
- For updated details on macOS and Windows compatibility, <a href="https://help.uaudio.com/hc/en-us/articles/210208926-Latest-Mac-Windows-Operating-System-Compatibility-Information" rel="noopener noreferrer" target="_blank">read this article</a>.
