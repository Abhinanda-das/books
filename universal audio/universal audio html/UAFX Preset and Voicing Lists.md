# UAFX Preset and Voicing Lists


This article lists all factory and artist presets and voicings that are available for UAFX pedals when using the UAFX Control mobile app. [Get UAFX Control](https://www.uaudio.com/pages/download-uafx-pedals).

## Notes

- The factory default presets for each pedal (and each effect on multi-effect pedals) are only available if you haven't loaded a preset from the app. You can save the factory default preset to the pedal so you can access it later by pressing the Preset switch, then you can store the preset by holding the Store switch. If you have already loaded a preset from the app, you can reset your pedal to recover the factory default presets. [**Back up your presets**](360058187832-UAFX-Control-Manual.html#h_01G3VKQ8QFRAEQV264M2MKE402) **before you reset the pedal!**
- When you load a Del-Verb Voicing, that Voicing is saved with the effect. Voicings are not editable and do not need to be saved or backed up.  

## About Custom Features in UAFX Factory Presets

- Factory presets marked with a checkmark (✓) indicate that the preset uses custom settings that are not available with stock pedal controls.
- After loading a custom factory preset, any custom controls that are available on knobs are maintained if controls are modified and stored as a new user preset. However, if you adjust a switch that is mapped to a custom setting, the stock settings for the switch are then used.
- To revert to all stock settings after loading a preset with custom settings, switch to Live mode and store a new preset.
