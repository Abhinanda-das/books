### UAD DSP Plug-Ins

UAD DSP plug-ins require **Apollo**, **Satellite**, or **UAD-2 PCIe** hardware to run.

All UAD DSP plug-ins are installed by the **UAD software installer**, which is found in the **Apollo** panel in UA Connect.

You can refer to the latest UAD Software compatibility information [**here**](210208926-Latest-macOS-Windows-Operating-System-Compatibility-Information.html).

<div class="callout callout--warning">

**Note**: **UAD Software version 11.6.0** requires **macOS Big Sur 11** as a minimum operating system on Apple computers. For complete release history information, visit the <a href="215270403-UAD-Version-History-Release-Notes.html" rel="noopener noreferrer" target="_blank"><strong>UAD Version History & Release Notes</strong></a>.

</div>

#### UAD-2 PCIe System Requirements

- Windows 10 or 11 (64-bit editions)

<!-- -->

- <span style="font-weight: 400;">macOS Big Sur 11, Monterey 12, Ventura 13, Sonoma 14, Sequoia 15 and Tahoe 26</span>

<!-- -->

- **Note**: **UAD Software version 11.6.0** requires **macOS Big Sur 11 **as a minimum operating system on Apple computers.
- Available PCI Express slot for each UAD-2 card
- Internet connection to download software and authorize UAD plug-ins
- Compatible plug-in DAW host software
- 25 gigabytes of available storage
