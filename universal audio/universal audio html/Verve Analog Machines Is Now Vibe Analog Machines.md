# Verve Analog Machines Is Now Vibe Analog Machines


Starting on **April 14, 2026**, the plug-in formerly known as **Verve Analog Machines** has been renamed to **Vibe Analog Machines**.\
\
This change only affects the product name—its features, functionality, and user experience remain the same. All existing users can continue using the product as usual. The plug-in will now appear in the DAW as **Vibe Analog Machines,** and the sonics of existing sessions will not be affected in any way. Any references in documentation or other materials now refer to Vibe Analog Machines. 

This also applies to the Essentials version, now called **Vibe Analog Machines Essentials.**
