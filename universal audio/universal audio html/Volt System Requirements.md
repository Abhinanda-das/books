# Volt System Requirements


### macOS & Windows System Requirements

- <span style="font-weight: 400;">Available USB 2.0 (or higher) port</span>
- <span style="font-weight: 400;">USB cable (included). See </span><a href="https://help.uaudio.com/hc/en-us/articles/4409364723732-Volt-Cable-Guide" rel="noopener noreferrer" target="_blank"><span style="font-weight: 400;">Volt Cable Guide</span></a><span style="font-weight: 400;"> for more info.</span>
- <span style="font-weight: 400;">Intel, AMD, or Apple Silicon processor</span>
- <span style="font-weight: 400;">Windows 10 or Windows 11 (64-bit editions)</span>
- <span style="font-weight: 400;">macOS 11 Big Sur or higher</span>
- <span style="font-weight: 400;">Internet connection for software downloads</span><span style="font-weight: 400;"></span>

### iPad & iPhone System Requirements 

- <span style="font-weight: 400;">iPadOS / iOS 14 or newer</span>

### Windows Compatibility Notes

- <span style="font-weight: 400;">Windows ARM systems are not compatible with UA software; Volt hardware is untested.</span>
- <span style="font-weight: 400;">ASIO and WDM driver modes are compatible. ASIO is recommended when supported by the host application.</span>
- Connect to a port that uses an **Intel** USB host controller when possible. See <a href="https://help.uaudio.com/hc/en-us/articles/18734594593812" rel="noopener noreferrer" target="_blank"><strong>this article</strong></a> for additional details.
- <span style="font-weight: 400;">Volt is compatible with Windows 10 64-bit edition and Windows 11. Older Windows operating systems are untested. Although these operating systems may work, they are not supported.</span>
- Windows operating systems running on Virtual Machines are untested and not supported.

### Additional Compatibility Notes

- 'Hackintosh' and 'Mac computers' not made by Apple are not supported.
- Linux distributions may work with Volt as the devices are class compliant USB, but no UA software is available for Linux, and such configurations are not supported or tested.
