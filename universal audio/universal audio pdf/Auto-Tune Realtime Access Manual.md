## **What is Auto-Tune Realtime Access?** 

For twenty years, Auto-Tune has been the world standard for professional pitch correction, and the tool of choice for the most iconic vocal effect in popular music. 

Now, with Auto-Tune Realtime Access, we're proud to bring that same technology to the UAD platform, with a simple and intuitive interface. 

## **What Kind of Audio is Appropriate for Auto-Tune?** 

Auto-Tune requires a single well-isolated sound source such as a solo voice, or a single instrument playing one pitch at a time. Noise or extreme breathiness in vocal performance can sometimes lead to tracking errors, which can often be remedied by adjusting the ‚ÄãTracking‚Äã ‚Äãsetting.

## **Quick Start** 

Below is a quick overview of the Auto-Tune Realtime Access workflow. 

## **Step 1 - Open Auto-Tune  Realtime Access** 

Place Auto-Tune Realtime Access on an audio track in your DAW. For best pitch correction results, use it on a vocal track with only one singer, or an instrumental track that does not include chords or multiple pitches sounding at once. 

If you are using Auto-Tune Realtime Access with a UA audio interface, load it as an insert in the Console application for the lowest possible latency when tracking in real-time. 

## **Step 2 - Choose the Correct Key and Scale** 

Set the ‚ÄãKey and Scale‚Äã parameters to match the actual key and scale of your music. 

If you‚Äôre not sure what key your music is in, you can use the ‚ÄãAuto-Key‚Äã plug-in (sold separately) to automatically detect it and send that information to Auto-Tune Realtime Access. 

You can also use the ‚ÄãKeyboard‚Äã to customize your scale by turning individual notes on and off. 

## **Step 3 - Choose your Retune Speed and Humanize Settings** 

Set the ‚ÄãRetune Speed‚Äã to determine how quickly Auto-Tune Realtime Access tunes your audio to the target pitches. For a pronounced Auto-Tune Effect, set the Retune Speed to Fast. For more natural-sounding pitch correction, set it to Medium or Slow. 

For more natural-sounding pitch correction on sustained notes, turn up ‚ÄãHumanize‚Äã.

## **Retune Speed** 


![](<Auto-Tune Realtime Access Manual_assets/Auto-Tune_Realtime_Access_Manual.pdf-0006-03.png>)


Set the Retune Speed to determine how quickly Auto-Tune Realtime Access tunes your audio to the target pitches.

## **Humanize** 


![](<Auto-Tune Realtime Access Manual_assets/Auto-Tune_Realtime_Access_Manual.pdf-0006-07.png>)


The Humanize function allows you to add realism to sustained notes by preserving subtle variations in pitch. 

Humanize applies a slower Retune Speed only during the sustained portion of longer notes. If the pitch of a sustained note sounds unnaturally static, set Humanize to Minimum or Maximum to reintroduce subtle pitch deviations from the original recording.

## **Keyboard** 


![](<Auto-Tune Realtime Access Manual_assets/Auto-Tune_Realtime_Access_Manual.pdf-0007-05.png>)


The Keyboard displays the current detected pitch by highlighting it in blue, and also allows you to add and remove notes from the scale. 


![](<Auto-Tune Realtime Access Manual_assets/Auto-Tune_Realtime_Access_Manual.pdf-0007-07.png>)


When a note on the Keyboard is On, it will appear white or black (depending on which note it is), and input pitches that are closest to that note will be tuned to it. 


![](<Auto-Tune Realtime Access Manual_assets/Auto-Tune_Realtime_Access_Manual.pdf-0007-09.png>)


When a note on the Keyboard is set to Off, it will appear grey, and any incoming pitches that are closest to that note will be tuned to the next closest scale note instead. 


![](<Auto-Tune Realtime Access Manual_assets/Auto-Tune_Realtime_Access_Manual.pdf-0007-11.png>)


When a note on the Keyboard is displayed in blue, that indicates the current detected pitch.

## **Enable Auto-Key Detection**

## **Show Tooltips** 

Tooltips are helpful hints that pop up when you hover over one of the controls in Auto-Tune Realtime Access. If you don‚Äôt want to see them, you can turn them off here. 

## **Choosy Tracking** 

In most cases, this should be left at its default setting of enabled. Try disabling it if audio is noisy or poorly isolated and pitch correction becomes unreliable.

Check this box before clicking the Save button to set the default settings for any new instances of Auto-Tune Realtime Access.
