# Пение — основные службы ISKCON

Карта папки. Пословный (каждое санскр./бенгали слово = русское значение) и литературный перевод основных служб. Все тексты сверены по авторитетным источникам (kksongs.org, ISKCON Desire Tree, vedabase, Krishna Balaram Mandir, Back to Godhead).

## Утренняя программа

- [[Шри Шри Гурв-аштака]] — *самсара-даванала* · Вишванатха Чакраварти Тхакур · санскрит · мангала-арати, брахма-мухурта
- [[Шри Гуру-пуджа (Гуру-вандана)]] — *шри-гуру-чарана-падма* · Нароттама даса Тхакур · бенгали · поклонение гуру
- [[Шри Туласи-пуджа]] — *намо намах туласи* · Кришнадаса Кавираджа Госвами · пранама + арати + обхождение + сбор листьев

## Вечерняя программа

- [[Шри Гаура-арати]] — *джая джая горачандер* · Бхактивинода Тхакур · бенгали · сандхья-арати

## Поётся после каждого арати

- [[Шри Нрисимха-пранама]] — *намас те нарасимхайа* · стихи 1–2 + Джаядева Госвами (Дашаватара-стотра)

## Структура каждого файла

Каждая служба оформлена единообразно: фронтматтер (автор, книга-источник, язык, источники, теги) → краткая справка о месте в службе → стихи. Под каждым стихом: текст в кириллической транслитерации (IAST-схема), пословный разбор, литературный перевод.
